String str = "ab" 和 String str = new String("ab"); 的区别

展开查看
  • 前者先检查字符串常量池中是否存在相同值的字符串，存在就直接返回对象，不存在就新创建一个对象保存在字符串常量池
  • 后者在堆区创建新的字符串对象，即使字符串常量池中有相同值的对象。

transient关键字的作用

展开查看 - 被修饰的对象不参与序列化 - 在ArrayList源码中有，元素存放在elementData变量，这个变量被transient修饰，elementData是一个缓存数组，它通常会预留一些容量，等容量不足时再扩充容量，那么有些空间可能就没有实际存储元素，采用上诉的方式来实现序列化时，就可以保证只序列化实际存储的那些元素，而不是整个数组，从而节省空间和时间。

jvm、jdk、jre的区别

展开查看
  • jvm是java虚拟机，是java程序运行的地方
  • jdk是java开发工具，包含了jre和编译java源码的编译器javac等工具
  • jre是java运行环境，包含java程序运行时所需的内容，包括jvm和基础类库

进程和线程

进程是资源分配的基本单位，线程是cpu调度的基本单位。

同类的多个线程共享进程的堆和方法区资源，但每个线程有自己的程序计数器、虚拟机栈和本地方法栈，所以系统在产生一个线程，或是在各个线程之间作切换工作时，负担要比进程小得多。
